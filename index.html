I'm a Full Stack Ruby on Rails Developer

I'm the bowtie. My wife, Mariane, is the wedding dress. We got married in August 2016. To our sides are my childhood best friend and his girly-pie.

I started developing HTML & CSS back in my teen years. In December 2015, a friend, Karson Hatch, called me up and said "Hey, another guy and I are thinking of doing a web developer course. Wanna do it too?" I said "Sure, what do I have to lose?"

So, we attended the DevPoint Labs 11-week, full-time, full stack Ruby on Rails boot camp. We had no idea what we were in for, and I LOVED IT! Between the pre-course work, the day-to-day with our instructors Jake Sorce and Dave Jungst, I learned a lot and, most importantly, I learned to love to learn programming. Since then, I've dived into more and various JS libraries, C# w/ Razor Syntax, etc.

TL;DR:

I'm a developer and only plan on moving forward.



Projects



Working on the new release of the Eclipse software suite for InfinityELS, I helped create a scalable, dynamic user interface for use by hundreds of small to big business customers utilizing the Eclipse software for customer and financial management. I'd provide a link for interaction, however, use of the software is proprietary to InfinityELS and its customers.

In the same Eclipse software project, I had the task of creating reports and C# controller methods to return JSON from a SQL database via Linq syntax. These methods included some language for limiting the range of values across the x-axis and specifying which values are represented at all on the JavaScript graphs that I created from the ChartJS library. I also developed models that returned values from the database for each customer's information through Razor syntax depending on which report was being produced.
